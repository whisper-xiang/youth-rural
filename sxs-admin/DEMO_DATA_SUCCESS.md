## 🎭 演示场景

### 1. 项目申报流程
```
学生创建项目（草稿）→ 提交审核（待审核）→ 学院审批（学院已审）→ 校级审批（已立项）→ 开展实践 → 提交成果 → 项目结项
```

### 2. 审批管理流程
```
学院管理员审批本学院项目 → 校级管理员审批通过项目 → 生成审批记录 → 发送系统通知
```

### 3. 评优管理流程
```
创建评优活动 → 专家为项目评分 → 计算总分 → 生成排名 → 公示结果
```
